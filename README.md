# Age Calculator

A professional web application that calculates a person's exact age.

## Features

- Calculate age in years, months, and days
- Total months lived
- Total weeks lived
- Total days lived
- Next birthday calculation
- Zodiac sign
- Day of birth

## Technologies Used

- HTML5
- CSS3
- JavaScript
